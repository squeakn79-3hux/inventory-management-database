# Reflection

This project helped me understand how a database project moves from a business scenario into a working system.

The most useful parts were designing the ERD, normalising the data and then seeing the database working through forms, queries and reports. It also showed me that the interface matters. The database can be solid, but if the front end is buggy, the user experience suffers.

If I built this again, I would keep the MySQL database idea but replace LibreOffice Base with a web interface.
