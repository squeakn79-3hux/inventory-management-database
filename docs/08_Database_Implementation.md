# Database Implementation

The database was implemented in **MySQL** and exported as a SQL dump.

## Database
- Database name: `brightsparkdb`
- Export file: `database/BrightSparks_Database.sql`
- MySQL dump: Complete table structure and data export

## Tables
- `category_table`
- `inventory_table`
- `product_table`
- `sales_table`
- `sold_relationship_table`
- `supplier_table`

LibreOffice Base was used to create the user interface. This allowed forms, queries and reports to be created on top of the MySQL database.

The implementation included standard SQL operations such as insert, update, delete and select, along with reports for low stock and top-selling products.
