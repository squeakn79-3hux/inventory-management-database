# Future Development

If this system was rebuilt or expanded, I would add:

- A web-based interface.
- Secure login and user authentication.
- Barcode scanning.
- Automatic low-stock alerts.
- A dashboard for sales and inventory.
- Better reporting filters.
- Cloud backup.
- Mobile or tablet support.

These improvements would make the system more useful in a real retail environment.
