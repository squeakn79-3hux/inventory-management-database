# Executive Summary

The Inventory Management Database System was created for **Bright Sparks Ltd**, a fictional Auckland electronics retailer. The purpose of the project was to replace manual stock and sales processes with a structured database that could store product, supplier, inventory and sales information in one place.

The database was built using **MySQL** and connected to a working interface in **LibreOffice Base**. It includes tables, relationships, forms, queries, reports and role-based permissions. The project shows practical database design, SQL implementation and testing.

The system is compact, but it solves a real business problem: keeping better control of inventory and producing useful sales and stock information when needed.
