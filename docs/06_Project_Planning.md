# Project Planning

The project followed a simple development path:

1. Review the business scenario.
2. Identify the required data.
3. Design the ERD.
4. Normalise the data from UNF through to 3NF.
5. Build the MySQL database.
6. Add records and test SQL operations.
7. Connect LibreOffice Base.
8. Build forms, queries and reports.
9. Test the final database.

The planning focused on building a working database first, then proving it worked through screenshots and testing evidence.
