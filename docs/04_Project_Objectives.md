# Project Objectives

The main objective was to design and build a working inventory database for Bright Sparks Ltd.

The project aimed to:

- Design a relational database from business requirements.
- Create an ERD and normalised table structure.
- Build the database in MySQL.
- Add sample data.
- Create forms, queries and reports in LibreOffice Base.
- Include role-based permissions.
- Test the database to confirm it worked correctly.
